To represent a 0D object in 2D space, there are 2 equations that need to be satisfied in order to check whether the point \[(A_{0}, A_{1})\] in 2D space lies on the 0D object defined by the point(s) \[(\alpha_{0,0}, \alpha_{0,1})\] which all lie on that 0D object.

Equation 1 to be satisfied: \[A_0 - 1.0 = 0\]
Equation 2 to be satisfied: \[A_1 - 2.0 = 0\]
