To represent a 1D object in 4D space, there are 3 equations that need to be satisfied in order to check whether the point \[(A_0, ..., A_{3})\] in 4D space lies on the 1D object defined by the point(s) \[(\alpha_{0,0},...,\alpha_{0,3}), (\alpha_{1,0},...,\alpha_{1,3}) \] which all lie on that 1D object.

Equation 1 to be satisfied: \[(A_{0} - \alpha_{0,0})[\alpha_{1,1} - \alpha_{0,1}] - (A_{1} - \alpha_{0,1})[\alpha_{1,0} - \alpha_{0,0}] = 0\]
Equation 2 to be satisfied: \[(A_{3} - \alpha_{0,3})[\alpha_{1,1} - \alpha_{0,1}] - (A_{1} - \alpha_{0,1})[\alpha_{1,3} - \alpha_{0,3}] = 0\]
Equation 3 to be satisfied: \[(A_{2} - \alpha_{0,2})[\alpha_{1,1} - \alpha_{0,1}] - (A_{1} - \alpha_{0,1})[\alpha_{1,2} - \alpha_{0,2}] = 0\]
