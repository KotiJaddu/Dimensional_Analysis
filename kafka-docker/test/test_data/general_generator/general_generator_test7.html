To represent a 2D object in 4D space, there are 2 equations that need to be satisfied in order to check whether the point \[(A_0, ..., A_{3})\] in 4D space lies on the 2D object defined by the point(s) \[(\alpha_{0,0},...,\alpha_{0,3}), ..., (\alpha_{2,0},...,\alpha_{2,3})\] which all lie on that 2D object.

Equation 1 to be satisfied: \[(A_{0} - \alpha_{0,0})[(\alpha_{1,1} - \alpha_{0,1})[\alpha_{2,2} - \alpha_{0,2}] - (\alpha_{1,2} - \alpha_{0,2})[\alpha_{2,1} - \alpha_{0,1}] ] - (A_{1} - \alpha_{0,1})[(\alpha_{1,0} - \alpha_{0,0})[\alpha_{2,2} - \alpha_{0,2}] - (\alpha_{1,2} - \alpha_{0,2})[\alpha_{2,0} - \alpha_{0,0}] ] + (A_{2} - \alpha_{0,2})[(\alpha_{1,0} - \alpha_{0,0})[\alpha_{2,1} - \alpha_{0,1}] - (\alpha_{1,1} - \alpha_{0,1})[\alpha_{2,0} - \alpha_{0,0}] ] = 0\]
Equation 2 to be satisfied: \[(A_{3} - \alpha_{0,3})[(\alpha_{1,1} - \alpha_{0,1})[\alpha_{2,2} - \alpha_{0,2}] - (\alpha_{1,2} - \alpha_{0,2})[\alpha_{2,1} - \alpha_{0,1}] ] - (A_{1} - \alpha_{0,1})[(\alpha_{1,3} - \alpha_{0,3})[\alpha_{2,2} - \alpha_{0,2}] - (\alpha_{1,2} - \alpha_{0,2})[\alpha_{2,3} - \alpha_{0,3}] ] + (A_{2} - \alpha_{0,2})[(\alpha_{1,3} - \alpha_{0,3})[\alpha_{2,1} - \alpha_{0,1}] - (\alpha_{1,1} - \alpha_{0,1})[\alpha_{2,3} - \alpha_{0,3}] ] = 0\]
